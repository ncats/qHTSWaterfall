plotWaterfallR Documentation

Plots 3D qHTS waterfall plot, given Pubchem activity file or NCATS qHTS format file.

Description

Plots 3D qHTS waterfall plot, given Pubchem activity file or NCATS qHTS format file.

Usage

plotWaterfall(
  inputFile,
  activityReadouts = c("Activity"),
  logMolarConcVector,
  pointColors = c("darkgreen", "royalblue3"),
  curveColors = c("darkgreen", "royalblue3"),
  inactiveColor = "gray",
  alpha = 1,
  pointSize = 2,
  plotInactivePoints = T,
  curveResolution = 25,
  plotAspectRatio = c(1, 1, 3)
)

Arguments

inputFile

An optional input file path. If none is provided, a file chooser will prompt.

activityReadouts

Activity data readouts to include in plot

logMolarConcVector

An optional input list of log molar concentrations, if input is not from PubChem.

pointColors

color list for activity readout data points. The order should match the activityReadoutsList order.

curveColors

color list for activity readout fitted dose-response curves. The order should match the activityReadoutsList order.

inactiveColor

color to display inactive data, default is 'gray'.

alpha

alpha transparency of the the plot lines, default is 1.0

pointSize

relative size of plotted points, default = 2

plotInactivePoints

TRUE will plot inactive data as datapoints, FALSE Will hide inactive data.

curveResolution

value between 25 and 250, number of points to define dose-response curves. Fewer points renders as connected straight lines.

plotAspectRatio

relative sizes of concentration axis (x), response axis (y), and waterfall width (z). Input as list, derault: c(1, 1, 3)

Examples

## Not run: 

# specify sample data file
filePath <- system.file("extdata", "NCATS_CMT1A_PMP22_Follow_Up.csv", package="qHTSWaterfall")

# supply log molar concentrations (for non-PubChem data only)
logConc <- c(
 -9.011761791,
 -8.534640544,
 -8.057519287,
 -7.580398031,
 -7.103276777,
 -6.626155522,
 -6.149034267,
 -5.671913012,
 -5.194791758,
 -4.717670503,
 -4.240549248
)

# make plotting call with desired parameter settings
# 3D Plot will be presented in a separate window.
plotWaterfall(
  inputFile=filePath,
  activityReadouts = c("fluc", "nluc"),
  logMolarConcVector = logConc,
  pointColors = c("darkgreen", "royalblue3"),
  curveColors = c("darkgreen", "royalblue3"),
  inactiveColor = "gray",
  pointSize = 3,
  alpha = 1,
  plotInactivePoints = F,
  curveResolution = 100,
  plotAspectRatio = c(2,2,5)
  )

## End(Not run)
